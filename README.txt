Для начала пройдемся по файловой структуре:
1. "account_info.xlsx" - эксель файл, куда будут добавляться данные о зарегистрированных аккаунтах: ПОЧТА - ПАРОЛЬ - СТАТУС (SUCCESS/FAILURE)
2. "Email + password_2.txt" - файл с почтой и паролем вида ЛОГИН:ПАРОЛЬ
3. "main.py" - основной файл работы скрипта
4. "Proxy.txt" - файл с проксями вида ХОСТ:ПОРТ:ЛОГИН:ПАРОЛЬ
5. "README.txt" - файл с инструкцией (этот файл)
6. "requirements.txt" - файл с зависимостями pip. там находятся все модули Пайтон, используемые в проекте
7. "chromedriver.exe" - хром браузер для работы скрипта 

Запуск и работа скрипта:
1. Заполняем файлы под номерами 2 и 4
2. Открываем консоль (терминал) и прописываем pip install -r requirements.txt
3. Запускаем файл main.py

Особенности работы:
!!! Если работа скрипта прервётся (или вы принудительно её прервёте), то в папке проекта может появиться zip архив. Следите за этим, и удаляйте если он появляется.

Рекомендуется регистрировать пачками по не более 50 аккаунтов.
Работает с почтами rambler.ru (и поддоменами rambler), с другими может не работать.
